# Lists in Markdown

## Ordered List

1. First item
2. Second item
3. Third item

## Unordered List

- First item
- Second item
- Third item

## Mixed List

1. First item
    - Subitem 1
    - Subitem 2
2. Second item
    - Subitem 1
      1. Sub-subitem 1
      2. Sub-subitem 2
    - Subitem 2
3. Third item
